Spam Classification

Joseph Fallait

Classification of spam is a common problem that email filters need to optimize for, as the price and speed of automatically generated spam has increase greatly throughout the years. Companies face greater overhead ever year dealing wth this spam outreach, so improving spam filters is a vital advancement in the modern tech ecosystem. This project used Naive Bayes and Support Vector Machine classification to accomplish this task using publicly accessible data, comparing their time performance.
